PLEASE FOLLOW THE INSTRUCTIONS ON HOW TO SETUP

STEP 1: IMPORT THE THE DATABASE ON MYSQL HEAD TO THE IMPORT SECTION TO IMPORT THE WEBSITE DATABASE.

IF YOU TRY TO ACCESS THE WEBSITE AND EXPERIENCE ITS FUNCTIONALITY OF THE WEBSITE 
YOU NEED TO CREATE A DUMMY ACCOUNT WITH THE EMAIL YOU USED TO RECEIVE OTP. 


STEP 2: CLICK THE DATABASE THAT YOU JUST IMPORTED EARLIER. YOU CAN SEE TABLES, 
YOU JUST INSERT THE YOUR DATA  FOR THE FOLLOWING TABLES LIST HERE!
*TABLE STUDENT
*TABLE GUEST
*TABLE EMPLOYEE

STEP 3: CHOOSE A TABLE AND INSERT YOUR DUMMY DATA  ON THE SECTION YOU WILL 
SEE ON THE TOP PORTION "INSERT" CLICK THAT MENU, AND INSERT THE FOLLOWING FORM AND CLICK GO, TO INSERT YOUR DATA.

STEP 4: HEAD TO THE WEBSITE AND YOU CAN NOW TRY THE WEBSITE.

HOPE YOU WILL ENJOY THE USER EXPERINCE AND THE WEBSITE FUNCTIONALITY!!
